# Network Availability & Uptime Monitor

A production-ready Bash automation script designed to monitor system endpoint availability, track server uptime, and detect network timeouts or infrastructure failures.

 Features
Automated Ping Sweeps: Sends controlled ICMP packets to verify remote host responsiveness.
Real-time Alerting: Simulates system monitoring warnings with automated timestamps for quick incident response.
Lightweight Performance: Optimized for Linux cron jobs to enable continuous background infrastructure monitoring.

 Tech Stack
Language: Bash (Shell Scripting)
Core Concepts: Network Troubleshooting, ICMP/Ping, Automation, Linux CLI, Infrastructure Monitoring
